# IT-Automation-Toolkit
PowerShell-based IT automation toolkit for Windows administration, system monitoring, security auditing, and network troubleshooting.

This project provides a collection of ready-to-use PowerShell scripts that help system administrators and IT professionals automate common administrative and diagnostic tasks.

---

## Features

### Windows Administration Scripts

- System Information Collection
- Event Log Analysis
- Performance Monitoring
- Service Audit
- Security Compliance Check
- User and Group Audit
- Windows Update Status Check

### Network Troubleshooting Scripts

- Ping Testing
- DNS Resolution Testing
- Port Availability Testing
- Traceroute Diagnostics
- Network Monitoring
- Internet Connectivity Check
- Network Configuration Collection
- Automated Report Generation
